# 🚀 LaunchIQ — AI Startup Intelligence Simulator

LaunchIQ is an AI-powered startup intelligence platform that helps entrepreneurs analyze, validate, and simulate startup ideas using multi-agent AI systems, competitor intelligence, geolocation analysis, market trends, and automated business report generation.

The platform combines AI agents, workflow orchestration, geospatial APIs, and interactive analytics to provide hyper-personalized startup insights and business recommendations.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/apekshakor/LauncIQ.git
cd LauncIQ
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
GROQ_API_KEY=your_api_key
OPENCAGE_API_KEY=your_api_key
```

---

# 🚀 Running the Application

```bash
streamlit run app.py
```

# 🔄 How LaunchIQ Works

1. User enters startup details
2. OpenCage API processes the location
3. Nearby competitors are discovered
4. AI agents analyze:

   * Market trends
   * Competition
   * Supply chain
   * Marketing opportunities
5. Workflow graph coordinates agent communication
6. Final startup intelligence report is generated
7. User downloads the professional PDF report
